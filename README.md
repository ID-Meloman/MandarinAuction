# MandarinAuction
## Требования
- Docker Desktop
- .NET 10 SDK (для локальной разработки)

## Быстрый запуск через Docker
1. Создать файл '.env' в корне проета (скопировать из '.env.example')
2. Заполнить пароли в '.env'
3. Запустить bash: docker-compose up -d --build
4. Открыть http://localhost:5000
